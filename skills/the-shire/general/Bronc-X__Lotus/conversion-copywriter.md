# Conversion Copywriter

Use only when the user explicitly invokes `/conversion-copywriter`.

This skill writes marketing copy, landing-page copy, product-page copy, ads, CTAs, slogans, and conversion-focused website text. It is not for internal communications, technical docs, or generic explanatory prose.

Write like a sharp conversion strategist: concrete, credible, emotionally aware, and commercially useful. Avoid hype, empty slogans, and generic AI polish.

## Workflow

1. **Identify the Sale**
   - Product or offer
   - Target customer
   - Use case
   - Pain or desire
   - Purchase motivation
   - Trust proof
   - Desired action

2. **Ask or Assume**
   - If critical context is missing, ask at most 3 questions.
   - If the user wants speed, proceed with explicit assumptions.

3. **Position First**
   - State what the page is really selling.
   - State why the user should act now.
   - State the biggest conversion obstacle.

4. **Write Page-Ready Copy**
   - H1
   - Subheadline
   - Primary CTA
   - Secondary CTA
   - 3-5 core benefits
   - Pain section
   - Solution section
   - Proof/trust section
   - FAQ or objection handling
   - A/B headline variants when useful

## Copy Standards

- Specific beats abstract
- Outcomes beat features
- Proof beats adjectives
- Short sentences beat long explanations
- Customer language beats industry jargon
- Strong verbs beat decorative modifiers
- Concrete contrast beats vague differentiation

Avoid empty phrases such as:

- empower
- revolutionize
- next-generation
- all-in-one solution
- seamless experience
- cutting-edge
- ultimate
- comprehensive platform

Use them only when supported by specific evidence and rewritten into customer language.

## Quality Gate

Before final output, check:

- Every headline can communicate value by itself.
- Every CTA names a clear action.
- Every benefit connects to a user outcome.
- Every claim either has evidence or is phrased modestly.
- The copy does not sound like a template.
- The copy can be pasted into a real page without a theory lecture.
